Virtual Environments



Virtual environments can be easily transported from one hardware (machine) to another allowing people across different machines to have the same state things shared on their own machines. Technologies like Docker and Kubernetes allow us to containerize applications.

A possible actions is to use Anaconda and set up a virtual environment for each project for example, and also specify the Python version, Python packages and more.
